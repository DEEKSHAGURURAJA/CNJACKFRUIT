# 🔍 Web Server Fingerprinting Tool

Overview
This project is a secure web server fingerprinting tool that identifies server types and versions by analyzing HTTP/HTTPS responses using low-level socket programming.

 Architecture
- Client-Server Model
- TCP Socket Communication
- SSL Encryption Layer

 Features
- HTTP/HTTPS banner grabbing
- SSL secure communication
- Multi-threaded scanning
- Server detection (Apache, Nginx, etc.)
- Response time analysis
- OS guessing
- Result logging and visualization

Technologies Used
- Python
- Socket Programming
- SSL (TLS)
- Multithreading

How to Run

### Server
```bash
python server.py
